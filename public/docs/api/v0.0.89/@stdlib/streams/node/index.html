Node Streams

Standard library Node.js streams.

Usage

var streams = require( '@stdlib/streams/node' );

streams

Standard library Node.js streams.

var s = streams;
// returns {...}

The package contains the following streams:

  • debugStream( [options,] [clbk] ): transform stream for debugging stream pipelines.
  • inspectStream( [options,] clbk ): transform stream for inspecting streamed data.
  • joinStream( [options] ): transform stream which joins streamed data.
  • splitStream( [options] ): transform stream which splits streamed data.
  • transformStream( [options] ): transform stream.

Examples

var getKeys = require( '@stdlib/utils/keys' );
var streams = require( '@stdlib/streams/node' );

console.log( getKeys( streams ) );

getgid

Return the numeric group identity of the calling process.

Usage

var getgid = require( '@stdlib/process/getgid' );

getgid()

Returns the numeric group identity of the calling process.

var id = getgid();

Notes

  • The function only returns an integer group identity on POSIX platforms. For all other platforms (e.g., Windows and Android), the function returns null.
  • See getgid(2).

Examples

var getgid = require( '@stdlib/process/getgid' );

var gid = getgid();
console.log( 'gid: %d', gid );

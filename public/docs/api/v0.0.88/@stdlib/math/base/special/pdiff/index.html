pdiff

Return the positive difference between x and y.

Usage

var pdiff = require( '@stdlib/math/base/special/pdiff' );

pdiff( x, y )

Returns the positive difference between x and y if x < y; otherwise, returns 0.

var v = pdiff( 4.2, 3.14 );
// returns 1.06

v = pdiff( 3.14, 4.2 );
// returns +0.0

v = pdiff( -0.0, +0.0 );
// returns +0.0

If any argument is NaN, the function returns NaN.

var v = pdiff( 4.2, NaN );
// returns NaN

v = pdiff( NaN, 3.14 );
// returns NaN

v = pdiff( NaN, NaN );
// returns NaN

Notes

  • This function is the equivalent of fdim in the C/C++ standard library.

Examples

var minstd = require( '@stdlib/random/base/minstd-shuffle' );
var pdiff = require( '@stdlib/math/base/special/pdiff' );

var x;
var y;
var v;
var i;

for ( i = 0; i < 100; i++ ) {
    x = minstd();
    y = minstd();
    v = pdiff( x, y );
    console.log( 'pdiff(%d,%d) = %d', x, y, v );
}

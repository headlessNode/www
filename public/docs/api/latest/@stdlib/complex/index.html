Complex Numbers

Complex number data types.

Usage

var ns = require( '@stdlib/complex' );

ns

Complex number data types.

var o = ns;
// returns {...}

The namespace contains the following sub-namespaces:

  • base: base (i.e., lower-level) complex number functions.
  • float32: single-precision complex floating-point number functions.
  • float64: double-precision complex floating-point number functions.

The namespace contains the following APIs:

  • complex( real, imag[, dtype] ): create a complex number.
  • ctors( dtype ): complex number constructors.
  • dtype( value ): return the data type of a complex number.
  • dtypes(): list of complex number data types.
  • promotionRules( [dtype1, dtype2] ): return the complex number data type with the smallest size and closest "kind" to which data types can be safely cast.
  • reviveComplex( key, value ): revive a JSON-serialized complex number.

Examples

var objectKeys = require( '@stdlib/utils/keys' );
var ns = require( '@stdlib/complex' );

console.log( objectKeys( ns ) );

Airplane From Above

Image of an airplane, viewed from above looking down.

Image of Fédèle Azari's gelatin silver print of an airplane, viewed from above looking down.


Usage

var image = require( '@stdlib/datasets/img-airplane-from-above' );

image()

Returns a buffer containing image data.

var img = image();
// returns <Buffer>

Examples

var image = require( '@stdlib/datasets/img-airplane-from-above' );

var img = image();
console.log( img );

CLI

Usage

Usage: img-airplane-from-above [options]

Options:

  -h,    --help                Print this message.
  -V,    --version             Print the package version.

Examples

$ img-airplane-from-above | <image_viewer>

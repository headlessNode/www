Locomotion

Twenty-four views of a house cat.

Image of Eadweard J. Muybridge's collotype of a house cat.


Usage

var image = require( '@stdlib/datasets/img-locomotion-house-cat' );

image()

Returns a buffer containing image data.

var img = image();
// returns <Buffer>

Examples

var image = require( '@stdlib/datasets/img-locomotion-house-cat' );

var img = image();
console.log( img );

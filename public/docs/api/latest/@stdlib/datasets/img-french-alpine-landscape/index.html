French Alpine Landscape

Image of a French alpine landscape.

Image of Adolphe Braun's carbon print of a French alpine landscape.


Usage

var image = require( '@stdlib/datasets/img-french-alpine-landscape' );

image()

Returns a buffer containing image data.

var img = image();
// returns <Buffer>

Examples

var image = require( '@stdlib/datasets/img-french-alpine-landscape' );

var img = image();
console.log( img );

CLI

Usage

Usage: img-french-alpine-landscape [options]

Options:

  -h,    --help                Print this message.
  -V,    --version             Print the package version.

Examples

$ img-french-alpine-landscape | <image_viewer>

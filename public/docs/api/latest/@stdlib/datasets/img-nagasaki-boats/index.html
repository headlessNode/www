Nagasaki Boats

Image of boats in a river in Nagasaki.

Image of Felice Beato's albumen silver print of boats in a river in Nagasaki.


Usage

var image = require( '@stdlib/datasets/img-nagasaki-boats' );

image()

Returns a buffer containing image data.

var img = image();
// returns <Buffer>

Examples

var image = require( '@stdlib/datasets/img-nagasaki-boats' );

var img = image();
console.log( img );

CLI

Usage

Usage: img-nagasaki-boats [options]

Options:

  -h,    --help                Print this message.
  -V,    --version             Print the package version.

Examples

$ img-nagasaki-boats | <image_viewer>

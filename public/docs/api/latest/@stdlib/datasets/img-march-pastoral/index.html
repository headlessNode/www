March Pastoral

Image of sheep in a pastoral setting.

Image of Peter Henry Emerson's photogravure of sheep in a pastoral setting.


Usage

var image = require( '@stdlib/datasets/img-march-pastoral' );

image()

Returns a buffer containing image data.

var img = image;
// returns <Buffer>

Examples

var image = require( '@stdlib/datasets/img-march-pastoral' );

var img = image();
console.log( img );

CLI

Usage

Usage: img-march-pastoral [options]

Options:

  -h,    --help                Print this message.
  -V,    --version             Print the package version.

Examples

$ img-march-pastoral | <image_viewer>

Acanthus mollis.

Image of Karl Blossfeldt's gelatin silver print Acanthus mollis.


Acanthus mollis is a plant with tall, straight stem and spiky, black tipped leaves.

Usage

var image = require( '@stdlib/datasets/img-acanthus-mollis' );

image()

Returns a buffer containing image data.

var img = image();
// returns <Buffer>

Examples

var image = require( '@stdlib/datasets/img-acanthus-mollis' );

var img = image();
console.log( img );

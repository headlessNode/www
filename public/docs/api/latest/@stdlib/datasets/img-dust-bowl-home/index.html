Abandoned Dust Bowl Home

Image of an abandoned Dust Bowl home.

Image of Dorothea Lange's gelatin silver print of an abandoned Dust Bowl home.


Usage

var image = require( '@stdlib/datasets/img-dust-bowl-home' );

image()

Returns a buffer containing image data.

var img = image();
// returns <Buffer>

Examples

var image = require( '@stdlib/datasets/img-dust-bowl-home' );

var img = image();
console.log( img );

CLI

Usage

Usage: img-dust-bowl-home [options]

Options:

  -h,    --help                Print this message.
  -V,    --version             Print the package version.

Examples

$ img-dust-bowl-home | <image_viewer>

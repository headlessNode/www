Incremental Machine Learning

Standard library incremental machine learning algorithms.

Usage

var ns = require( '@stdlib/ml/incr' );

ns

Standard library incremental machine learning algorithms.

var incr = ns;
// returns {...}
  • incrBinaryClassification( N[, options] ): incrementally perform binary classification using stochastic gradient descent (SGD).
  • incrkmeans( k[, ndims][, options] ): incrementally partition data into k clusters.
  • incrSGDRegression( [options] ): online regression via Stochastic Gradient Descent.

Bernoulli distribution cumulative distribution function.

where 0 <= p <= 1 is the success probability.

Usage

var cdf = require( '@stdlib/stats/base/dists/bernoulli/cdf' );

cdf( x, p )

Evaluates the cumulative distribution function for a Bernoulli distribution with success probability p.

var y = cdf( 1.0, 0.5 );
// returns 1.0

y = cdf( 0.5, 0.5 );
// returns 0.5

If provided NaN as any argument, the function returns NaN.

var y = cdf( NaN, 0.5 );
// returns NaN

y = cdf( 0.0, NaN );
// returns NaN

If provided a success probability p outside of [0,1], the function returns NaN.

var y = cdf( 0.9, -1.0 );
// returns NaN

y = cdf( 0.9, 1.5 );
// returns NaN

cdf.factory( p )

Returns a function for evaluating the cumulative distribution function of a Bernoulli distribution with success probability p

var mycdf = cdf.factory( 0.5 );
var y = mycdf( 0.5 );
// returns 0.5

y = mycdf( 2.0 );
// returns 1.0

Examples

var randu = require( '@stdlib/random/base/randu' );
var cdf = require( '@stdlib/stats/base/dists/bernoulli/cdf' );

var p;
var x;
var y;
var i;

for ( i = 0; i < 10; i++ ) {
    x = randu() * 5.0;
    p = randu();
    y = cdf( x, p );
    console.log( 'x: %d, p: %d, F(x;p): %d', x.toFixed( 4 ), p.toFixed( 4 ), y.toFixed( 4 ) );
}

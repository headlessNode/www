Variance

Bernoulli distribution variance.

where p is the success probability.

Usage

var variance = require( '@stdlib/stats/base/dists/bernoulli/variance' );

variance( p )

Returns the variance of a Bernoulli distribution with success probability p.

var v = variance( 0.1 );
// returns ~0.09

v = variance( 0.5 );
// returns 0.25

If provided a success probability p outside of [0,1], the function returns NaN.

var v = variance( NaN );
// returns NaN

v = variance( 1.5 );
// returns NaN

v = variance( -1.0 );
// returns NaN

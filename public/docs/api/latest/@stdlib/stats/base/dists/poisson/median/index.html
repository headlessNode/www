Poisson distribution median.

where λ is the mean parameter.

Usage

var median = require( '@stdlib/stats/base/dists/poisson/median' );

median( lambda )

Returns the median of a Poisson distribution with mean parameter lambda.

var v = median( 9.0 );
// returns 9

v = median( 0.5 );
// returns 0

If provided lambda < 0, the function returns NaN.

var v = median( -1.0 );
// returns NaN

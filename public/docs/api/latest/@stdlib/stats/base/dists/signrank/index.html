Signed Rank

Distribution of the Wilcoxon signed rank test statistic.

Usage

var signrank = require( '@stdlib/stats/base/dists/signrank' );

signrank

Distribution of the Wilcoxon signed rank test statistic.

var dist = signrank;
// returns {...}

The namespace contains the following distribution functions:

  • cdf( x, n ): wilcoxon signed rank test statistic cumulative distribution function.
  • pdf( x, n ): wilcoxon signed rank test statistic probability density function.
  • quantile( p, n ): wilcoxon signed rank test statistic quantile function.

Examples

var objectKeys = require( '@stdlib/utils/keys' );
var signrank = require( '@stdlib/stats/base/dists/signrank' );

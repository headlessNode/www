Skewness

Normal distribution skewness.

The skewness for a normal random variable with mean μ and standard deviation σ > 0 is

Usage

var skewness = require( '@stdlib/stats/base/dists/normal/skewness' );

skewness( mu, sigma )

Returns the skewness for a normal distribution with parameters mu (mean) and sigma (standard deviation).

var y = skewness( 2.0, 1.0 );
// returns 0.0

y = skewness( -1.0, 4.0 );
// returns 0.0

If provided NaN as any argument, the function returns NaN.

var y = skewness( NaN, 1.0 );
// returns NaN

y = skewness( 0.0, NaN );
// returns NaN

If provided sigma <= 0, the function returns NaN.

var y = skewness( 0.0, 0.0 );
// returns NaN

y = skewness( 0.0, -1.0 );
// returns NaN

Examples

var randu = require( '@stdlib/random/base/randu' );
var skewness = require( '@stdlib/stats/base/dists/normal/skewness' );

var sigma;
var mu;
var y;
var i;

for ( i = 0; i < 10; i++ ) {
    mu = ( randu()*10.0 ) - 5.0;
    sigma = randu() * 20.0;
    y = skewness( mu, sigma );
    console.log( 'µ: %d, σ: %d, skew(X;µ,σ): %d', mu.toFixed( 4 ), sigma.toFixed( 4 ), y.toFixed( 4 ) );
}

Standard Deviation

Lognormal distribution standard deviation.

The standard deviation for a lognormal random variable is

where μ is the location parameter and σ > 0 is the scale parameter. According to the definition, the natural logarithm of a random variable from a lognormal distribution follows a normal distribution.

Usage

var stdev = require( '@stdlib/stats/base/dists/lognormal/stdev' );

stdev( mu, sigma )

Returns the standard deviation for a lognormal distribution with location mu and scale sigma.

var y = stdev( 2.0, 1.0 );
// returns ~15.969

y = stdev( 0.0, 1.0 );
// returns ~2.161

y = stdev( -1.0, 2.0 );
// returns ~19.901

If provided NaN as any argument, the function returns NaN.

var y = stdev( NaN, 1.0 );
// returns NaN

y = stdev( 0.0, NaN );
// returns NaN

If provided sigma <= 0, the function returns NaN.

var y = stdev( 0.0, 0.0 );
// returns NaN

y = stdev( 0.0, -1.0 );
// returns NaN

Examples

var randu = require( '@stdlib/random/base/randu' );
var stdev = require( '@stdlib/stats/base/dists/lognormal/stdev' );

var sigma;
var mu;
var y;
var i;

for ( i = 0; i < 10; i++ ) {
    mu = ( randu()*10.0 ) - 5.0;
    sigma = randu() * 20.0;
    y = stdev( mu, sigma );
    console.log( 'µ: %d, σ: %d, SD(X;µ,σ): %d', mu.toFixed( 4 ), sigma.toFixed( 4 ), y.toFixed( 4 ) );
}

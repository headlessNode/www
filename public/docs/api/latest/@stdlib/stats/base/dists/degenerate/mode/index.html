Mode

Degenerate distribution mode.

The mode for a degenerate random variable is

where μ is the constant value of the distribution.

Usage

var mode = require( '@stdlib/stats/base/dists/degenerate/mode' );

mode( mu )

Returns the mode of a degenerate distribution with constant value mu.

var v = mode( 10.0 );
// returns 10.0

v = mode( -0.5 );
// returns -0.5

Examples

var randu = require( '@stdlib/random/base/randu' );
var mode = require( '@stdlib/stats/base/dists/degenerate/mode' );

var mu;
var v;
var i;

for ( i = 0; i < 10; i++ ) {
    mu = randu();
    v = mode( mu );
    console.log( 'µ: %d, Mode(X;µ): %d', mu.toFixed( 4 ), v.toFixed( 4 ) );
}

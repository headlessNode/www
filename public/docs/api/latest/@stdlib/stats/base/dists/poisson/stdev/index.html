Standard Deviation

Poisson distribution standard deviation.

The standard deviation for a Poisson random variable is

where λ is the mean parameter.

Usage

var stdev = require( '@stdlib/stats/base/dists/poisson/stdev' );

stdev( lambda )

Returns the standard deviation of a Poisson distribution with mean parameter lambda.

var v = stdev( 9.0 );
// returns 3.0

v = stdev( 0.5 );
// returns ~0.707

If provided lambda < 0, the function returns NaN.

var v = stdev( -1.0 );
// returns NaN

Examples

var randu = require( '@stdlib/random/base/randu' );
var round = require( '@stdlib/math/base/special/round' );
var stdev = require( '@stdlib/stats/base/dists/poisson/stdev' );

var lambda;
var v;
var i;

for ( i = 0; i < 10; i++ ) {

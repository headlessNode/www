Median

Student's t distribution median.

The median for a Student's t random variable with degrees of freedom Î½ is

Usage

var median = require( '@stdlib/stats/base/dists/t/median' );

median( v )

Returns the median of a Student's t distribution with degrees of freedom v.

var y = median( 9.0 );
// returns 0.0

y = median( 1.5 );
// returns 0.0

If provided v < 0, the function returns NaN.

var y = median( -1.0 );
// returns NaN

Examples

var randu = require( '@stdlib/random/base/randu' );
var round = require( '@stdlib/math/base/special/round' );
var median = require( '@stdlib/stats/base/dists/t/median' );

var v;
var y;
var i;

for ( i = 0; i < 10; i++ ) {
    v = randu() * 20.0;
    y = median( v );
    console.log( 'v: %d, Median(X,v): %d', v.toFixed( 4 ), y.toFixed( 4 ) );
}

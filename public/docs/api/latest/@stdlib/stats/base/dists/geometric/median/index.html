Geometric distribution median.

where p is the success probability.

Usage

var median = require( '@stdlib/stats/base/dists/geometric/median' );

median( p )

Returns the median of a geometric distribution with success probability p.

var v = median( 0.1 );
// returns 6

v = median( 0.5 );
// returns 0

If provided a success probability p outside of [0,1], the function returns NaN.

var v = median( NaN );
// returns NaN

v = median( 1.5 );
// returns NaN

v = median( -1.0 );
// returns NaN

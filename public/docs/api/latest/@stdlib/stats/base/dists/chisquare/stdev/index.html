Standard Deviation

Chi-squared distribution standard deviation.

The standard deviation for a chi-squared random variable is

where k >= 0 is the degrees of freedom.

Usage

var stdev = require( '@stdlib/stats/base/dists/chisquare/stdev' );

stdev( k )

Returns the standard deviation of a chi-squared distribution with degrees of freedom k.

var v = stdev( 9.0 );
// returns ~4.243

v = stdev( 0.5 );
// returns 1.0

If provided k < 0, the function returns NaN.

var v = stdev( -1.0 );
// returns NaN

Examples

var randu = require( '@stdlib/random/base/randu' );
var round = require( '@stdlib/math/base/special/round' );
var stdev = require( '@stdlib/stats/base/dists/chisquare/stdev' );

var v;
var i;

Assert

Base double-precision floating-point number assert functions.

Usage

var assert = require( '@stdlib/number/float64/base/assert' );

assert

Namespace containing "base" double-precision floating-point number assert functions.

var ns = assert;
// returns {...}

The namespace contains the following functions:

  • isSameValueZero( a, b ): test if two double-precision floating-point numbers are the same value.
  • isSameValue( a, b ): test if two double-precision floating-point numbers are the same value.

Examples

var objectKeys = require( '@stdlib/utils/keys' );
var ns = require( '@stdlib/number/float64/base/assert' );

console.log( objectKeys( ns ) );

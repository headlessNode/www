Uint32

Base utilities for unsigned 32-bit integers.

Usage

var ns = require( '@stdlib/number/uint32/base' );

ns

Base utilities for unsigned 32-bit integers.

var o = ns;
// returns {...}
  • fromBinaryStringUint32( bstr ): create an unsigned 32-bit integer from a literal bit representation.
  • rotl32( x, shift ): bitwise rotation to the left.
  • rotr32( x, shift ): bitwise rotation to the right.
  • toBinaryStringUint32( x ): return a string giving the literal bit representation of an unsigned 32-bit integer.
  • uint32ToInt32( x ): convert an unsigned 32-bit integer to a signed 32-bit integer.

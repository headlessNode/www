Euler's Number

Usage

var e = require( '@stdlib/constants/float64/e' );

e

The mathematical constant e, also known as Euler's number or Napier's constant. e is the base of the natural logarithm.

var bool = ( e === 2.718281828459045 );
// returns true

Examples

var e = require( '@stdlib/constants/float64/e' );

console.log( e );
// => 2.718281828459045

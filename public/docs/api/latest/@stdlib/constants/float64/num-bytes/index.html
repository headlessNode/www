Number of Bytes

Size (in bytes) of a double-precision floating-point number.

Usage

var FLOAT64_NUM_BYTES = require( '@stdlib/constants/float64/num-bytes' );

FLOAT64_NUM_BYTES

Size (in bytes) of a double-precision floating-point number.

var bool = ( FLOAT64_NUM_BYTES === 8 );
// returns true

Examples

var FLOAT64_NUM_BYTES = require( '@stdlib/constants/float64/num-bytes' );

console.log( FLOAT64_NUM_BYTES );
// => 8

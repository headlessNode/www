Constants

8-bit unsigned integer mathematical constants.

Usage

var constants = require( '@stdlib/constants/uint8' );

constants

8-bit unsigned integer mathematical constants.

var c = constants;
// returns {...}
  • MAX: maximum unsigned 8-bit integer.
  • NUM_BYTES: size (in bytes) of an 8-bit unsigned integer.

Examples

var objectKeys = require( '@stdlib/utils/keys' );
var constants = require( '@stdlib/constants/uint8' );

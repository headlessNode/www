Number of Bytes

Size (in bytes) of a single-precision floating-point number.

Usage

var FLOAT32_NUM_BYTES = require( '@stdlib/constants/float32/num-bytes' );

FLOAT32_NUM_BYTES

Size (in bytes) of a single-precision floating-point number.

var bool = ( FLOAT32_NUM_BYTES === 4 );
// returns true

Examples

var FLOAT32_NUM_BYTES = require( '@stdlib/constants/float32/num-bytes' );

console.log( FLOAT32_NUM_BYTES );
// => 4

String Path Constants

Standard library string path constants.

Usage

var constants = require( '@stdlib/constants/path' );

constants

Standard library string path constants.

var ns = constants;
// returns {...}
  • DELIMITER_POSIX: POSIX path delimiter.
  • DELIMITER_WIN32: Windows path delimiter.
  • DELIMITER: platform-specific path delimiter.
  • SEP_POSIX: POSIX path segment separator.
  • SEP_WIN32: Windows path segment separator.
  • SEP: platform-specific path segment separator.

Examples

var objectKeys = require( '@stdlib/utils/keys' );
var constants = require( '@stdlib/constants/path' );

console.log( objectKeys( constants ) );

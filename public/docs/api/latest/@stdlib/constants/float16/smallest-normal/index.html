Smallest Normal

Smallest positive normalized half-precision floating-point number.

Usage

var FLOAT16_SMALLEST_NORMAL = require( '@stdlib/constants/float16/smallest-normal' );

FLOAT16_SMALLEST_NORMAL

Smallest positive normalized half-precision floating-point number.

// returns true

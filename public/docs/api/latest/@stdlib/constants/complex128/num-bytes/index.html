Number of Bytes

Size (in bytes) of a 128-bit complex number.

Usage

var COMPLEX128_NUM_BYTES = require( '@stdlib/constants/complex128/num-bytes' );

COMPLEX128_NUM_BYTES

Size (in bytes) of a 128-bit complex number.

var bool = ( COMPLEX128_NUM_BYTES === 16 );
// returns true

Examples

var COMPLEX128_NUM_BYTES = require( '@stdlib/constants/complex128/num-bytes' );

console.log( COMPLEX128_NUM_BYTES );
// => 16

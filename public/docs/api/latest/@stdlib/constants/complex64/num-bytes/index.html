Number of Bytes

Size (in bytes) of a 64-bit complex number.

Usage

var COMPLEX64_NUM_BYTES = require( '@stdlib/constants/complex64/num-bytes' );

COMPLEX64_NUM_BYTES

Size (in bytes) of a 64-bit complex number.

var bool = ( COMPLEX64_NUM_BYTES === 8 );
// returns true

Examples

var COMPLEX64_NUM_BYTES = require( '@stdlib/constants/complex64/num-bytes' );

console.log( COMPLEX64_NUM_BYTES );
// => 8

Number of Bytes

Size (in bytes) of an 8-bit signed integer.

Usage

var INT8_NUM_BYTES = require( '@stdlib/constants/int8/num-bytes' );

INT8_NUM_BYTES

Size (in bytes) of an 8-bit signed integer.

var bool = ( INT8_NUM_BYTES === 1 );
// returns true

Examples

var INT8_NUM_BYTES = require( '@stdlib/constants/int8/num-bytes' );

console.log( INT8_NUM_BYTES );
// => 1

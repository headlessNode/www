Max Uint32

Maximum unsigned 32-bit integer.

Usage

var UINT32_MAX = require( '@stdlib/constants/uint32/max' );

UINT32_MAX

Maximum unsigned 32-bit integer. 

var bool = ( UINT32_MAX === 4294967295 );
// returns true

Notes

  • UINT32_MAX is the maximum array length.

Examples

var UINT32_MAX = require( '@stdlib/constants/uint32/max' );

console.log( UINT32_MAX );
// => 4294967295

Number of Bytes

Size (in bytes) of a 32-bit unsigned integer.

Usage

var UINT32_NUM_BYTES = require( '@stdlib/constants/uint32/num-bytes' );

UINT32_NUM_BYTES

Size (in bytes) of a 32-bit unsigned integer.

var bool = ( UINT32_NUM_BYTES === 4 );
// returns true

Examples

var UINT32_NUM_BYTES = require( '@stdlib/constants/uint32/num-bytes' );

console.log( UINT32_NUM_BYTES );
// => 4

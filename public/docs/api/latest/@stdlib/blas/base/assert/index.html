Assert

Base BLAS assertion utilities.

Usage

var ns = require( '@stdlib/blas/base/assert' );

ns

Base BLAS assertion utilities.

var o = ns;
// returns {...}
  • isDiagonalType( value ): test if an input value is a BLAS diagonal type.
  • isLayout( value ): test if an input value is a BLAS memory layout.
  • isMatrixTriangle( value ): test if an input value is a BLAS matrix triangle.
  • isOperationSide( value ): test if an input value is a BLAS operation side.
  • isTransposeOperation( value ): test if an input value is a BLAS transpose operation.

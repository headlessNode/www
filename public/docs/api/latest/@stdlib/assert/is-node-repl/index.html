isNodeREPL

Check if running in a Node.js REPL environment.

Usage

var isNodeREPL = require( '@stdlib/assert/is-node-repl' );

isNodeREPL()

Returns a boolean indicating if running in a Node.js REPL environment.

var bool = isNodeREPL();
// returns <boolean>

Examples

var isNodeREPL = require( '@stdlib/assert/is-node-repl' );

console.log( isNodeREPL() );
// => <boolean>

IS_BROWSER

Check if the runtime is a web browser.

Usage

var IS_BROWSER = require( '@stdlib/assert/is-browser' );

IS_BROWSER

Boolean indicating if the runtime is a web browser.

var bool = IS_BROWSER;
// returns <boolean>

Notes

  • In order to determine whether the runtime is a web browser, the implementation must resolve the global scope, thus requiring function generation. The use of function generation may be problematic in browser contexts enforcing a strict content security policy (CSP).

Examples

var IS_BROWSER = require( '@stdlib/assert/is-browser' );

console.log( IS_BROWSER );
// => <boolean>

IS_ELECTRON

Check if the runtime is Electron.

Usage

var IS_ELECTRON = require( '@stdlib/assert/is-electron' );

IS_ELECTRON

Boolean indicating if the runtime is Electron.

var bool = IS_ELECTRON;
// returns <boolean>

Examples

var IS_ELECTRON = require( '@stdlib/assert/is-electron' );

console.log( IS_ELECTRON );
// => <boolean>

See Also

  • @stdlib/assert/is-electron-main: check if the runtime is the main Electron process.
  • @stdlib/assert/is-electron-renderer: check if the runtime is the Electron renderer process.

isWebWorker

Check if the runtime is a web worker.

Usage

var IS_WEB_WORKER = require( '@stdlib/assert/is-web-worker' );

IS_WEB_WORKER

Boolean indicating if the runtime is a web worker.

var bool = IS_WEB_WORKER;
// returns <boolean>

Notes

  • In order to determine whether the runtime is a web worker, the implementation must resolve the global scope, thus requiring function generation. The use of function generation may be problematic in browser contexts enforcing a strict content security policy (CSP).

Examples

var IS_WEB_WORKER = require( '@stdlib/assert/is-web-worker' );

console.log( IS_WEB_WORKER );
// => <boolean>

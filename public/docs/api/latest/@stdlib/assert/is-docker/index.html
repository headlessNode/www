IS_DOCKER

Check if the process is running in a Docker container.

Usage

var IS_DOCKER = require( '@stdlib/assert/is-docker' );

IS_DOCKER

Boolean indicating if the process is running in a Docker container.

var bool = IS_DOCKER;
// returns <boolean>

Examples

var IS_DOCKER = require( '@stdlib/assert/is-docker' );

console.log( IS_DOCKER );
// => <boolean>

isArray

Test if a value is an array.

Usage

var isArray = require( '@stdlib/assert/is-array' );

isArray( value )

Tests if a value is an array.

var bool = isArray( [] );
// returns true

Examples

var isArray = require( '@stdlib/assert/is-array' );

var bool = isArray( [ 1, 2, 3, 4 ] );
// returns true

bool = isArray( {} );
// returns false

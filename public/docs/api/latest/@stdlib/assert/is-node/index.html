IS_NODE

Check if the runtime is Node.js.

Usage

var IS_NODE = require( '@stdlib/assert/is-node' );

IS_NODE

Boolean indicating if the runtime is Node.js.

var bool = IS_NODE;
// returns <boolean>

Notes

  • In order to determine whether the runtime is Node.js, the implementation must resolve the global scope, thus requiring function generation. The use of function generation may be problematic in browser contexts enforcing a strict content security policy (CSP).

Examples

var IS_NODE = require( '@stdlib/assert/is-node' );

console.log( IS_NODE );
// => <boolean>

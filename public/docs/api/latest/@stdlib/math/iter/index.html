Math Iterators

Standard library math iterators.

Usage

var ns = require( '@stdlib/math/iter' );

ns

Standard library math iterators.

var iterators = ns;
// returns {...}

The namespace contains the following sub-namespaces:

  • ops: standard library math operator iterators.
  • sequences: standard library math iterators for generating sequences.
  • special: standard library math iterators for special functions.

Examples

var objectKeys = require( '@stdlib/utils/keys' );
var ns = require( '@stdlib/math/iter' );

console.log( objectKeys( ns ) );

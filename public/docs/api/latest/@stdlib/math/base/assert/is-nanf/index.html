isnanf

Test if a single-precision floating-point numeric value is NaN.

Usage

var isnanf = require( '@stdlib/math/base/assert/is-nanf' );

isnanf( x )

Tests if a single-precision floating-point numeric value is NaN.

var bool = isnanf( NaN );
// returns true

Examples

var isnanf = require( '@stdlib/math/base/assert/is-nanf' );

var bool = isnanf( NaN );
// returns true

bool = isnanf( 5.0 );
// returns false

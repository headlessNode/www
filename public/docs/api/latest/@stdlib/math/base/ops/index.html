Operators

Standard library base math operators.

Usage

var ns = require( '@stdlib/math/base/ops' );

ns

Standard library base math operators.

var operators = ns;
// returns {...}

The namespace contains the following functions:

  • cadd( [out,] re1, im1, re2, im2 ): add two complex numbers.
  • cdiv( [out,] re1, im1, re2, im2 ): divide two complex numbers.
  • cmul( [out,] re1, im1, re2, im2 ): multiply two complex numbers.
  • cneg( [out,] re, im ): negate a complex number.
  • csub( [out,] re1, im1, re2, im2 ): subtract two complex numbers.

Examples

var objectKeys = require( '@stdlib/utils/keys' );
var ns = require( '@stdlib/math/base/ops' );

console.log( objectKeys( ns ) );

Hacovercosine

Compute the half-value coversed cosine.

The half-value coversed cosine is defined as

Usage

var hacovercos = require( '@stdlib/math/base/special/hacovercos' );

hacovercos( x )

Computes the half-value coversed cosine (in radians).

var v = hacovercos( 0.0 );
// returns 0.5

v = hacovercos( 3.141592653589793/2.0 );
// returns 1.0

v = hacovercos( -3.141592653589793/6.0 );
// returns 0.25

Examples

var linspace = require( '@stdlib/array/linspace' );
var TWO_PI = require( '@stdlib/constants/float64/two-pi' );
var hacovercos = require( '@stdlib/math/base/special/hacovercos' );

var x = linspace( 0.0, TWO_PI, 100 );
var i;

for ( i = 0; i < x.length; i++ ) {
    console.log( hacovercos( x[ i ] ) );
}

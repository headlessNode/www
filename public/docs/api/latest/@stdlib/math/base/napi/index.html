napi

C APIs for registering a Node-API module exporting interfaces.

Usage

var ns = require( '@stdlib/math/base/napi' );

ns

Namespace for C APIs for registering a Node-API module exporting interfaces.

var napi = ns;
// returns {...}

The namespace contains the following packages:

  • binary( env, info, fcn ): C APIs for registering a Node-API module exporting interfaces for invoking binary numerical functions.
  • ternary( env, info, fcn ): C APIs for registering a Node-API module exporting interfaces for invoking ternary numerical functions.
  • unary( env, info, fcn ): C APIs for registering a Node-API module exporting interfaces for invoking unary numerical functions.

Examples

var objectKeys = require( '@stdlib/utils/keys' );
var ns = require( '@stdlib/math/base/napi' );

console.log( objectKeys( ns ) );

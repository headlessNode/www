utils

Standard library basic mathematical utilities.

var o = utils;
// returns {...}
  • absdiff( x, y ): compute the absolute difference of two real numbers.
  • epsdiff( x, y[, scale] ): compute the relative difference of two real numbers in units of double-precision floating-point epsilon.
  • reldiff( x, y[, scale] ): compute the relative difference of two real numbers.

Slice

Slice namespace.

Usage

var ns = require( '@stdlib/slice' );

ns

Slice namespace.

var o = ns;
// returns {...}

The namespace contains the following:

  • base: base slice namespace.
  • Slice(): slice constructor.
  • MultiSlice( ...slice ): multi-slice constructor.
  • seq2slice( str, len, strict ): convert a subsequence string to a Slice object.

Examples

var objectKeys = require( '@stdlib/utils/keys' );
var ns = require( '@stdlib/slice' );

console.log( objectKeys( ns ) );

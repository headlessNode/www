minutesInYear

Determine the number of minutes in a year according to the Gregorian calendar.

Usage

var minutesInYear = require( '@stdlib/time/minutes-in-year' );

minutesInYear( [value] )

Returns the number of minutes in a year according to the Gregorian calendar.

var num = minutesInYear();
// returns <number>

By default, the function returns the number of minutes in the current year (according to local time). To determine the number of minutes for a particular year, provide either a year or a Date object.

var num = minutesInYear( new Date() );
// returns <number>

num = minutesInYear( 2000 );
// returns 527040

num = minutesInYear( 2017 );
// returns 525600

Notes

  • The function's return value is a generalization and does not take into account inaccuracies due to daylight savings conventions, crossing timezones, or other complications with time and dates.

Examples

var minutesInYear = require( '@stdlib/time/minutes-in-year' );

var v;
var i;

for ( i = 0; i < 2021; i++ ) {
    v = minutesInYear( i );
    console.log( 'The year %d has %d minutes.', i, v );
}

CLI

Usage

Usage: minutes-in-year [options] [year]

Options:

  -h,    --help                Print this message.
  -V,    --version             Print the package version.

Examples

$ minutes-in-year
<number>

For a specific year,

$ minutes-in-year 2016
527040

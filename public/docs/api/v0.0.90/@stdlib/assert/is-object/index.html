isObject

Test if a value is an object.

Usage

var isObject = require( '@stdlib/assert/is-object' );

isObject( value )

Tests if a value is an object.

var bool = isObject( {} );
// returns true

bool = isObject( true );
// returns false

Notes

  • The function returns false if provided an array or null.

    var bool = isObject( [] );
// returns false

bool = isObject( null );
// returns false

Examples

var isObject = require( '@stdlib/assert/is-object' );

var bool = isObject( {} );
// returns true

bool = isObject( new Date() );
// returns true

bool = isObject( /.*/ );
// returns true

bool = isObject( null );
// returns false

bool = isObject( [] );
// returns false

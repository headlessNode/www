Big Endian

Check if an environment is big endian.

Usage

var IS_BIG_ENDIAN = require( '@stdlib/assert/is-big-endian' );

IS_BIG_ENDIAN

Boolean indicating if an environment is big endian.

var bool = IS_BIG_ENDIAN;
// returns <boolean>

Examples

var IS_BIG_ENDIAN = require( '@stdlib/assert/is-big-endian' );

console.log( IS_BIG_ENDIAN );
// => <boolean>

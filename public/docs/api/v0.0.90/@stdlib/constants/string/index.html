String Constants

Standard library string constants.

Usage

var constants = require( '@stdlib/constants/string' );

constants

Standard library string constants.

var ns = constants;
// returns {...}
  • PATH_DELIMITER_POSIX: POSIX path delimiter.
  • PATH_DELIMITER_WIN32: Windows path delimiter.
  • PATH_DELIMITER: platform-specific path delimiter.
  • PATH_SEP_POSIX: POSIX path segment separator.
  • PATH_SEP_WIN32: Windows path segment separator.
  • PATH_SEP: platform-specific path segment separator.
  • UNICODE_MAX_BMP: maximum Unicode code point in the Basic Multilingual Plane (BMP).
  • UNICODE_MAX: maximum Unicode code point.

Examples

var objectKeys = require( '@stdlib/utils/keys' );
var constants = require( '@stdlib/constants/string' );

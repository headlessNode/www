Math Iterators

Standard library math iterators.

Usage

var ns = require( '@stdlib/math/iter' );

ns

Standard library math iterators.

var iterators = ns;
// returns {...}

The namespace contains the following functions for creating iterator protocol-compliant iterators:

  • iterAdd( iter0, ...iterator ): create an iterator which performs element-wise addition of two or more iterators.
  • iterDivide( iter0, ...iterator ): create an iterator which performs element-wise division of two or more iterators.
  • iterFibonacci( [options] ): create an iterator which generates a Fibonacci sequence.
  • iterMod( iter0, ...iterator ): create an iterator which performs an element-wise modulo operation of two or more iterators.
  • iterMultiply( iter0, ...iterator ): create an iterator which performs element-wise multiplication of two or more iterators.
  • iterNonFibonacci( [options] ): create an iterator which generates a non-Fibonacci integer sequence.
  • iterSubtract( iter0, ...iterator ): create an iterator which performs element-wise subtraction of two or more iterators.

Examples

var objectKeys = require( '@stdlib/utils/keys' );
var ns = require( '@stdlib/math/iter' );

console.log( objectKeys( ns ) );
